## Running Locally

### Clone the repository
```sh
git clone https://github.com/CapivaraComputerClub/ccc
cd ccc
```

### Start the server

```sh
# if using yarn
yarn start

# if using npm
npm run start

# directly with Node
node src/index.mjs

```
